# mortgage-calc

Калькулятор: купить трёшку в ипотеку сейчас или копить на вкладе?

Открыть `index.html` в браузере — двигать ползунки — получить ответ.

## Метрики
- **NPV** — выгода в деньгах сегодня (+ = ипотека выгоднее)
- **MIRR** — годовая доходность сделки (сравниваем со ставкой вклада)
- **ΔW** — разница итогового капитала через N лет
